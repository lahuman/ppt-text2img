# Release Notes

## v1.1 - 2026-04-13

### 요약
`v1.1`은 사용성 개선에 초점을 맞춘 업데이트입니다. 데스크톱 UI를 더 보기 쉽게 다듬고, 변환 진행 상황이 실제로 보이도록 수정했으며, 변환 전에 놓치기 쉬운 주의 사항을 앱 안에서 자연스럽게 확인할 수 있도록 보강했습니다.

### 주요 변경 사항
- 데스크톱 UI를 전면 개선했습니다.
- 한국어/영어 UI 전환을 정리하고 안내 문구를 강화했습니다.
- 변환 전 확인 팝업과 상시 주의 사항 패널을 추가했습니다.
- 진행 상태 텍스트와 진행 바가 변환 중 정상적으로 갱신되도록 수정했습니다.
- 긴 파일명을 선택해도 진행 영역이 가려지지 않도록 레이아웃을 개선했습니다.
- 그룹 내부 텍스트도 놓치지 않도록 그룹 해제 처리 흐름을 반영했습니다.
- Windows/PowerPoint/`pywin32` 실행 환경 오류를 더 명확한 메시지로 안내합니다.
- 소개 페이지와 문서를 최신 UI 기준으로 업데이트했습니다.

### 사용자 영향
- 배포용 PPT를 만들 때 현재 진행 상태를 더 쉽게 파악할 수 있습니다.
- PowerPoint 종료 여부나 파일 잠금 같은 실수로 인한 실패 가능성을 줄였습니다.
- 한국어/영어 사용자가 모두 더 자연스럽게 프로그램을 사용할 수 있습니다.

### 사용 전 확인 사항
- Windows 10 / 11 환경에서 실행해야 합니다.
- Microsoft PowerPoint가 설치되어 있어야 합니다.
- 변환 전에 Microsoft PowerPoint를 완전히 종료하세요.
- 변환 대상 PPT 파일이 열려 있지 않은지 확인하세요.
